How it works: Upload Excel (.xlsx, .xls) or CSV (.csv) files. Each Excel file can have multiple sheets with any column names. The bot will search all uploaded data when answering questions. If AI is enabled, the data is sent as context to the AI. If AI is off or unavailable, the bot searches the data directly.

Tips: Use descriptive column headers. Include columns like "Question", "Answer", "Product", "Category", "Description" etc. All sheets and columns are automatically indexed. Admin-uploaded files apply to everyone instantly.
